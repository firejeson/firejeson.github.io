一点点前期需要准备的工作awa

1.注册github账号

需要梯子

打开github GitHub

在中间白色候选框输入邮箱点击sign up for github，之后按照流程完成注册。

不建议使用以数字开头的qq邮箱（以及其他数字开头邮箱）。

gayhubgithub上多为英文，建议稍微熟悉一下操作逻辑等，周内可能会再出一篇进阶教程，其中有一些更深入的教程。

2.安装python

大概也许不是很需要梯子

打开python网页 Welcome to Python.org

网页内第二导航栏选择Downloads-Windows-最新版本（目前为3.11.0）

等待文件下载完成后打开。

一定要勾选下面的Add python.exe to PATH！！

如需更改安装路径，选择Customize……

安装完成后WIn+R打开运行窗口，输入cmd打开命令行提示符，键入python后回车如出现一下页面即代表已成功加入PATH。

如有需要，可以安装其他基于python的工具如you-get（下载b站视频）ffmpeg（转码）等。

如果因为网络原因无法下载，可选择国内镜像包阿里巴巴开源镜像站-OPSX镜像站-阿里云开发者社区 (aliyun.com)

但是这样下载有点小难度

3.安装VS Code

大概可能还是需要梯子的

访问Visual Studio Code - Code Editing. Redefined

点那块蓝蓝的，stable build。下载的慢的话看下面的国内镜像。

下载后配置可以看看这篇文章史上最全vscode配置使用教程 - 知乎 (zhihu.com)

绝对不是因为懒不想手打

进入扩展页面搜索并安装以下几个扩展（如果在上面那篇文章没安装的话）

对Python提供支持

对MarkDown文件提供支持（.md结尾，本文就是markdown)

以下几个是可选的

对c++提供支持

对c++直接编译提供支持

对Arduino单片机开发提供支持

就这样吧，剩下的拓展玩意下次再写。
